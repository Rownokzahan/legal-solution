Flexbox is a one-dimensional layout tool for aligning items and distributing space along a single axis. Grid is a two-dimensional layout tool for creating complex and responsive grid-based layouts. Flexbox is good for small-scale layouts, while Grid is better for larger-scale layouts.

Media query is a CSS technique that allows you to apply different styles to a webpage based on the characteristics of the device it is being viewed on, such as screen size, resolution, and screen orientation. They are used to make web pages responsive and adapt to different screen sizes and devices.

The CSS box model is the way elements on a webpage are laid out and sized. It defines the space an element takes up, including the element's content, padding, borders, and margins. The size of an element is determined by the width and height properties, along with the box-sizing property, which determines whether the element's size includes padding and borders or not.

Semantic tags are HTML elements that provide meaning to the structure of the content on a webpage. They describe the type of content contained within the element, such as headings, paragraphs, lists, images, etc. These tags help search engines and assistive technologies understand the content of the page and improve accessibility. Examples of semantic tags include <header> , <nav> , <main> , <article> , <section> , <aside> , <footer> , <figure> etc.
